# Godot Builds

This repository contains files for compiling Godot for various platforms using
[GitLab CI](https://about.gitlab.com/features/gitlab-ci-cd/).

Currently built:
- HTML5 (WebAssembly + asm.js)
- Linux (64-bit only)
- Windows (64-bit + 32-bit)
